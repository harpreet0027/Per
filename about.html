About Assessing Dynamic Learners' Behaviour

E-leaning,today is recognized as a big platform for learning and is considered better as compared to the conventional classroom learning. Personalized e-learning I one of the most researched areas, because courses online require a proper structure such that it is created keeping in mind the students it is being created for.

For the theoretical evaluation of the learning style of the user, VAK Learning style questionare has been used. It consists of 30 questions and depending on the performance in the questionare the user is classified as a - Visual Learner, Auditory Learner or Kinesthetic Learner. On the other hand, for the practical evaluation of the user, the user is asked to first attempt an aptitude test which will examine the aptitude of the user and accordingly the user will be recommended to take up Basic Python Course or Advance Python course for the further assessment. If the user opts for the Basic Python Course, he will get access to the first three topics of the course in the form of text as well as the video. After reading the text content , the user will be asked to perform a quiz and the performance will be stored in the database. Similarly, after completing the video part he will be asked to perform a quiz and the performance will be stored in the database again. After repeating the same for the first three topics, the performance for text and video will be compared and the user will be recommended the better mode of learning for him -Text or Video. The rest of the course will be presented to him in the preferred mode. After completing the entire course, the user will have to perform a final evaluation in the form of quiz or assignment.
